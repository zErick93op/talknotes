# Landing Page Clone

Este proyecto es una réplica visual de una landing page existente realizada con fines educativos y de práctica.

## Objetivo

Practicar:

- HTML semántico
- CSS
- Responsive Design
- Flexbox y Grid
- Optimización de estilos
- Maquetación de interfaces modernas

## Aviso

Este proyecto fue desarrollado únicamente con fines de aprendizaje. El diseño original pertenece a sus respectivos autores/propietarios.

No existe ninguna afiliación con la empresa o producto original.

## Tecnologías

- HTML5
- CSS3
- JavaScript
